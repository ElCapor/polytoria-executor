# PolyHack Documentation

Welcome to the documentation repository for PolyHack, the open-source executor for Polytoria. This documentation is powered by [Zensical](https://zensical.org/).

##  Getting Started

To contribute to the documentation or preview your changes locally, follow these simple steps:

### Installation

Ensure you have Python installed, then run:

```bash
pip install zensical
```

### Local Development

To start a local preview server and test your changes:

```bash
zensical serve
```

This will host the documentation locally (typically at [http://127.0.0.1:8000](http://127.0.0.1:8000)), allowing you to see your edits in real-time.

##  How to Contribute

All documentation content is located in the [docs/](docs/) directory and written in Markdown.

1.  **Find the file:** Navigate to the relevant .md file in the [docs/](docs/) folder.
2.  **Edit:** Make your changes using your favorite Markdown editor.
3.  **Test:** Run `zensical serve` to ensure your changes look correct.
4.  **Submit:** Save your work and submit a Pull Request!

Thank you for helping improve PolyHack!
